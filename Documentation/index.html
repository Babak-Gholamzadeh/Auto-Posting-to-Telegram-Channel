Introduction

As we all know, Telegram is one of the best-known messengers in the world, so using it to attract users to our website is wise.

You can use Telegram channel for noticing the users of publishing posts in your website, but you don't have to spend your time to send each post yourself.

'Auto-Posting to Telegram Channel' is a WordPress plugin for sending website posts to a Telegram channel automatically. It has been designed for news, magazine, newspaper, interview, review, personal websites etc, to make you able to send your posts to your Telegram channel, as you publish them in the website.

Sending posts to your channel with this plugin is a piece of cake. Also simply you can customize your posts and add some features to them before sending to your channel.

In this documentation it's tried to explain all parts and options of the plugin.

Features

Send Post Elements

Send Title, Excerpt, Content and Tags of Posts and also 'Read More' as Hyperlink

Change Style of Elements

Change the Style of Them to Bold, Italic and Regular, or Even Hyperlink

Send Featured Image

Send the Featured Image of the Posts with Custom Position

Send Custom Text & Link

Send Your Custom Text and Hyperlink or a Signature

Send Previous Posts

Send Your Previous Posts Just with One Click

Update Channel Posts

Channel Posts will be Updated as You Update Website Posts

Pin a Post in Channel

Whenever a Post be Sent or Updated, Can be Pinned in Channel

Send Link Buttons

Send some Link Buttons at the Bottom of Posts

The Number of Visitors

Show the Number of Visitors who Come to Website through Channel

Set a Delay

Set a Delay for Sending Posts After Publishing on Website

Installation

In order to install the plugin, first you have to have installed WordPress (if you haven't installed it yet, go here to learn how to do that).

So we imagine you have installed WordPress and now you are going to install the plugin.


You have two options to install the plagin:

  1. Installing directly in WordPress Dashboard
  2. Using FTP access

And here both of them will be explained.


First: Installing directly in WordPress Dashboard

    Keep step by step:

  1. Unzip the file that you downloaded from Envato Market to the location that you desire
  2. After going to WordPress Dashboard, click on Plugins → Add New
  3. After that, click on Upload Plugins and then Browse...
  4. Via the File Upload dialog, choose ApTch.zip file (in the folder that you unzipped at the first step)
  5. Then click on Install Now
  6. After a few seconds as the message below be shown, you should click on Active Plugin
  7. Finally you will see 'Telegram Settings' is appeared in the menu (like the image below), which means the plugin has been installed and actived

Second: Using FTP access

    Keep step by step:

  1. Unzip the file that you downloaded from Envato Market to the location that you desire
  2. And then unzip the ApTch.zip file and only use the extracted ApTch plugin folder
  3. Log into your host server via an FTP software
  4. Upload the extracted ApTch plugin folder into wp-content → plugins folder
  5. In order to active the newly installed plugin, go to WordPress Dashboard and then Plugins → Installed Plugins
  6. Here you will see the plugin is right over there, but it's not actived yet. So in order to active the plugin just click on Active link under the plugin name
  7. Finally you will see the plugin settings menu is appeared (like the image below) which means the plugin has been installed and actived

Initialize Settings

Now the plugin is almost ready to be used, but first you must initialize the plugin in order to be able to communicate with your Telegram channel.

First of all, you need a Telegram bot. If you have already had a bot, you can skip this step.

Creating a Telegram Bot

Creating a bot in Telegram is so simple. You just need to go to your Telegram app and search for @BotFather there.

After you find the bot, START it and follow the steps below:

  1. After you start the bot, a list of commands will be displayed for you. You can click on the first command /newbot, or type this command yourself.
  2. Followed by running the command, the bot will ask you to choose a name for your bot.
  3. then it will ask you to enter a username for your bot, and the username must end in 'bot' in anyway (and certainly the username must be something unique).
  4. After you enter a unique username for your bot, a message will be shown that give you a token.

Now you have your own bot. But keep the username and token of your bot, because you will need them in the further steps.

Creating a Telegram Channel

Now you need to create a Telegram channel. If you have already had a channel, you can skip this step.

In order to create a channel just follow the steps below:

  1. First click on at the top-left corner of your Telegram app and then a side menu will be appeared.
  2. Now click on the second option New Channel, and a message box is shown that ask you to choose a name for your channel. After you enter the name click on CREATE.
  3. In the next step, it will ask you to choose a link for your channel (ensure that the first option be selected Public Channel), and certainly the link must be something unique.
  4. After that it will ask you to invite some members into your channel which you can skip that step.

Now you have your own channel. But keep the link of your channel, because you will need it in the further steps.

Adding the Bot as an Admin in the Channel

After having a bot and a channel, now you must add your bot in your channel as an administrator. If your bot has already been added to your channel as an admin with enough permissions (Post messages, Edit messages of others), you can skip this step.

For doing that, follow the steps below:

  1. First go to your channel, click on at the top-right corner of the channel.
    After clicking on that, a menu will be shown.
  2. Click on the third option that titled Manage Channel, and then select Administrators.
  3. After that, it will show the list of administrators of the channel, and for adding the bot, click on ADD ADMINISTRATOR.
  4. Here search for the username of your bot, do you remember that?
  5. After selecting your bot, a message will be displayed, just click on OK.
  6. In the final step you must set the permissions for the bot, and you need to mark at least Post messages & Edit messages of others, and then click on SAVE.
    Then the list of administrators will be shown again and this time your bot is there as well (just click on CLOSE and then CANCEL).

Good job! Now it is the time to go back to WordPress Dashboard to fill some fields in order to initialize the plugin.

Fill the Required Fields to Initialize the Plugin

Now in WordPress Dashboard go to Telegram Settings.

Here there are two fields that ask you to enter the Bot Token and the Channel ID (the public link) of yours.

After you fill the fields, click on Check & Save, and the plugin will check and save them.

Initializing the plugin is done! Now you need to customize the format of the posts that be displayed on your channel.

Customize Settings

In this section you must select the elements of the website posts to be shown on the channel posts.

Here you can also change the style of them and add some custom texts & hyperlinks. Even you can add some link buttons at the bottom of the posts, which all of them will be explained in detail.

Customize Tab Overview

First of all, let's go to Customize tab to see what is going on there.

Here you have some options for customizing the channel posts, and playing with them is fun.

Explaining the image above by the numbers:

  1. If this option be ON, By Default all the website posts will be sent to your channel.

    2. Tip: Whenever you want to publish a post in your website (or even update it), you have another option (that will be explained further) to enable sending post to the channel for each post. And this option here just plays the role as a default setting.

  2. This box here is just a simulator of your channel and the smaller box inside of it, is the posts of the channel.

Add/Remove Items

In order that the elements of the website posts (i.e., title, excerpt, content, tags etc of the posts) be shown on the channel posts, you must add each of them here as an item.

For example, if you want the title and the excerpt of website posts be sent to the channel, simply you just need to add them here.

OK, let's get into it to add some items.

  1. Hover your cursor over in the white box, and the color of add button will become green.

    2. Tip: You will see another add button at the bottom of the white box, now don't pay attention it, because that will be explained further in detail.

  2. After clicking on the add button, a menu will be shown that has some items.
    • Featured Image: It will add the Featured Image of website posts. This item will be explained further.
    • Title: It will add the Title of website posts.
    • Excerpt: It will add the Excerpt of website posts.
    • Content: It will add the Content of website posts.
    • Tags: It will add the Tags of website posts.
    • Read More: It will add a Hyperlink that be referred to the website post link.
    • Custom Text: It will add your custom Text or Hyperlink.
  3. Click on which item that you want. For example, here we start with adding the Title.
  4. In order to add another item, you must hover your cursor over the added items, and you will see two add button be appeared. The upper add button is for adding another item before this item, and the lower one is for adding after this item.
  5. As the same way you can add the other items.
    And removing an item is much simpler than adding it. Just click on

Tip: The items will be shown on the channel posts in the same order as shown here.

Changing the Styles and Options of the Items

After adding some items you might need to change the styles of them (just a few styles that be offered by Telegram).

First of all, in order to tweak them, you must hover your cursor over the added items.

The first option is for adding a break line or a few empty lines after an item. For doing that just click on

After each click on that, you will see the tiny number on the similar icon on the left side of that be increased.

And for decreasing that just click on the (-) beside of the number.

All the items have this option and all of them will go to the new line by default (which means that the tiny number for all of them will show '1' by default).

Tip: If you completely decrease that to zero (and it be disappeared), on the channel posts will be put just a space (' ') after that item, not any break line.

The next option is changing the style of items. here be explained each items one by one:

For accessing the other options of the items, after moving your mouse on them, click on that item and then the options will be slide down. (Click on the area is indicated in the image below)

  1. Title: You can change the style of the Title of the posts to Bold and Italic, and even make it as a Hyperlink that refers to the website post. Or maybe you want to leave it as Regular.
  2. Excerpt: For the Excerpt you have the same options as the Title item.
  3. Content: For the Content you have the Bold and Italic Styles, and also you have another option to determine the max length of the content to be shown on the channel posts. (The default value is 400 characters of the content)

    4. Tip: Telegram posts have a limitation of characters which be described in the next section (Featured Image).

  4. Tags: You are able to put the tags of the website posts in the channel posts as #Hashtag or even make them as Hyperlink that refer to their own link on the website. (By default they are put as Hashtag)
  5. Read More: For this item you can just change the label of the hyperlink that refers to the website post. (By default the label of that is 'Read More')
  6. Custom Text: You can put your custom text here that be shown on the channel posts and you can make it Bold and Italic.

    7. But if you select the Link option for this item, it will not refer to the website post. After you click on the Link option, another text-box will be appeared which you can enter your custom link there.

    Tip: If you only enter a text in this item, it will be shown on the channel as a plain text.
    If you only enter a link without any text, the link will be shown on the channel itself.
    But if you enter a text along with a link, it will be shown as a hyperlink.

    Important: Ensure to put a valid link (means http[s]://...), because Telegram refuses to send an invalid link and the posts might be Not sent.

Featured Image

You have another item that can be added to send Featured Image of the website posts.

In order to add this item just click on any add button in the white box (it doesn't matter which one of them), and then click on Featured Image

After that a pop-up message comes on and you must choose that you desire the image be shown Below the post text or Above the post text.

Important: The difference between them except its appearance, is because of the limitation of the characters that Telegram has for posts. So if the image be shown Above the text, you are able to send just 200 characters as the caption of the image. And if the image be shown Below the text, you are able to send up to 4096 characters.

After you choose the position of image, the item of that will be added.

But you should know that, after you add the Feature Image item, you can change its position just by moving your mouse on the item and click on or

After adding some items with a little tweaking, your customization will be looked something like the image below.

Important: Keep in mind, if the total characters of the post elements that are supposed to be sent to the channel be more than the limitation (200 characters for showing the image Above the text, and 4096 characters for showing the image Below the text), the post will Not be sent.

Link Buttons

After Adding the main elements of the posts, you might need to add some buttons at the bottom your posts that be referred to your custom link or the website post.

For doing that just hover your cursor over the bigger box, and you will see another add button will be appeared under the white box.

Just click on that to add a button.

Now you can see a button be added. For adding more buttons you have three options:

  1. Add another button in a new row before this row of button
  2. Add another button in a new row After this row of button
  3. Add another button in the same row and just beside of that button

For adding another button, hover your cursor over the added button and options for adding will be apeared.

Click on the upper add button to add a new row of button before this row.

Click on the lower add button to add a new row of button after this row.

And Click on the little add button on the right side of the previous button to add a new button in the same row.

In this way you can add buttons with any order that you want. (For example like image below)

Tip: There is a limitation for adding button in a row, and you are able to add only 8 buttons in each row. But there is no limitation for adding rows.

In order to remove a row of button, as you might guess, just click on

As you can see, the default caption for added buttons is 'Link'. In order to change their caption and also their link, click on each button and its options will be appeared.

By default it asks you to enter you custom link, but by selecting the Post Link you can set that button be referred to the website post.

But if you select Custom Link, you must put your link in its field.

Important: If you select Custom Link, Ensure to Not leave the link field empty and put a valid link (means http[s]://...), because Telegram refuses to send an invalid link and the posts might be Not sent.

Settings a Delay to Send Posts

Maybe you would like to set a Delay for sending posts to the channel after they be published on the website, for any reason.

For setting a Delay, click on at the top-right corner of the white box.

After clicking on that you can enter the delay in minute. (Enter 0 or leave it empty to unset the delay)

Important: Remember that if you set a delay, for instance 10 minutes, and your post be published at 12 o'clock on the website. So as you expect, that post be sent to the channel at 12:10 (which means this time will be scheduled to send the post). BUT WordPress will send that just when someone visits your webiste and the schedule time has passed.
Therefore, this option doesn't be recommended. just use it if you have to, or your website has many visitors.

Other Options

As you can remember, there is a option in the Customize tab for telling the plugin to send the website posts to the channel (titled Enable Auto-Posting), that is only a default setting (which means, the plugin will send all the website posts to the channel by default).

Additionally that option, there is another option in the editor page for sending/editing the posts, which titled 'Telegram Post Options'. Via that box you can tell the plugin whether to send this individual post to the channel.

If you be on editing a post, and that post has already been sent to the channel, the first option in that box will be titled Update This Post, and if you edited the website post, that post in the channel will be edited as well.

Tip: The Default value of the first option in that box (Send This Post/Update This Post) depends on the value of Enable Auto-Posting in the Customize tab in Telegram Settings menu.

Tip: If you want to update the post in the channel and that post has already been deleted from the channel, the post will be sent again, as a new post.

Tip: If you want to update the post in the channel in order to change the position of the image, that post in the channel will not be changed and this post will be sent to the channel as a new post.

The second option in that box which titled Pin This Post is used to pin this post in the channel after it be sent to the channel or be updated there. By pinning a post, it will be shown at the top of the channel.

The next option is in the Posts → All Posts that shows all of your published posts.

As you can see, a new column has been added here, which has Telegram icon

Here you can see that all the posts have a paper plane icon in their row

By clicking on each icon, its post will be sent to the channel. (This option is just a quick way to send the posts to the channel)

After clicking on them, a short loading will be displayed (like this ), and after just few seconds that post will be sent to the channel.

If the post be sent to the channel, that icon will be changed and shows the number zero, for now.

This is an exciting number that shows the number of visitors who have opened the link of that post through the channel and have visited the post on the website. which means this number will be increased by any visitor that comes to your website post via your channel. (LC is abbreviation for Link Click)
